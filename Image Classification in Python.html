Image Classification

Introduction

The dataset used is the Dog Vs Cats which is the collection of images of dogs and cats. The subset of images (1002 for the training set and 100 for the test set) is considered from the dataset consisting of 25000 images. The objective is to build models to classify whether images contain either a dog or a cat. We will be comparing various classification models (machine learning algorithms and deep learning algorithms) and compute the accuracy of models and thereby selecting the most accurate model after fine tuning it.

Various Steps involved:

  1. Preprocessing: Loading and displaying the images,resizing the images,normalisation of the image data, dimension reduction using PCA, Training and Validation splits
  2. Building Models: K-Nearest Neighbors, Support Vector Machine, Random Forest and Convolutional Neural Network (CNN)
  3. Comparison of the Accuracy Scores: Before and after applying dimension reduction using PCA
  4. Hyperparameter Tuning: Hyperparameter Tuning of the best model

Baseline machine learning models

We will now run the baseline machine learning models: K-Nearest Neighbors, Support Vector Machine, and Random Forest.

K-Nearest Neighbors: This algorithm determines the similarity between the new observation and existing ones and groups the new observation into the category that is most similar to the available categories by calculating the Euclidian distance.

Support Vector Machine: Support Vector Classifier are classes that can perform both binary and multi-class classification. The objective is of this algorithm is find the hyperplane in an N-dimensional space where N are the number of features that classifies the data.

Random Forest: Random Forest is a tree-based algorithm which creates forest of many trees.

Ten-fold cross validation was also performed on the validation set where the data is segmented into parts and use all but one part for training and the remaining one for testing.

def run_models(training_data, val_data, class_train, class_val):
    classifiers = [KNeighborsClassifier(),RandomForestClassifier(),SVC()]
    
    models = {"K-Nearest Neighbors": KNeighborsClassifier(),
             "Support Vector Machine": SVC(),"Random Forest": RandomForestClassifier()}
    
    performance_results = pd.DataFrame()
    for key,value in models.items():
        model = value.fit(training_data, class_train)
   
        # 10 Fold Cross Validation on Validation Data
        cv_accuracy_score = cross_val_score(estimator=value, X=val_data ,y=class_val,cv=10)
    
        model_results = pd.DataFrame([[key,round(cv_accuracy_score.mean()*100,2)]], columns = ['Model', 'Validation Set Accuracy'])
        performance_results = performance_results.append(model_results)

    return performance_results

Dimensionality Reduction using Principal Component Analysis (PCA)

Principal Component Analysis (PCA) technique is used to reduce the features of a large dataset to have less number of features which could explain most of the information that is in the data. PCA in images compresses the image vectors such that it has preserved most of the image data. Using PCA function from the sklearn library, we have given the parameter n_components = 0.9 which means that PCA function will reduce the numbers of features such that those reduced features would be able to explain 90 % of the imformation present in the image data.

In [36]:
def cnn_model(dropout_rate=0,learning_rate=0.01):
    
    # create model
    model1 = Sequential()
    model1.add(Conv2D(32, (3, 3), activation='relu', input_shape=(350, 350, 3)))
    model1.add(MaxPooling2D(pool_size=(2, 2)))
    model1.add(Dropout(dropout_rate))

    model1.add(Conv2D(64, (3, 3), activation='relu'))
    model1.add(MaxPooling2D(pool_size=(2, 2)))
    model1.add(Dropout(dropout_rate))

    model1.add(Conv2D(128, (3, 3), activation='relu'))
    model1.add(MaxPooling2D(pool_size=(2, 2)))
    model1.add(Dropout(dropout_rate))

    model1.add(Flatten())
    model1.add(Dense(64, activation='relu'))
    model1.add(Dropout(dropout_rate))
    model1.add(Dense(1, activation='sigmoid')) # 2 because we have cat and dog classes

    optimizer = keras.optimizers.Adam(learning_rate=learning_rate)
    model1.compile(loss='binary_crossentropy', optimizer = optimizer, metrics=['accuracy'])

    return model1

def tuned_cnn_model(dropout_rate=0.25,learning_rate=0.001):
    
    # create model
    tuned_model = Sequential()
    tuned_model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(350, 350, 3)))
    tuned_model.add(MaxPooling2D(pool_size=(2, 2)))
    tuned_model.add(Dropout(dropout_rate))

    tuned_model.add(Conv2D(64, (3, 3), activation='relu'))
    tuned_model.add(MaxPooling2D(pool_size=(2, 2)))
    tuned_model.add(Dropout(dropout_rate))

    tuned_model.add(Conv2D(128, (3, 3), activation='relu'))
    tuned_model.add(MaxPooling2D(pool_size=(2, 2)))
    tuned_model.add(Dropout(dropout_rate))

    tuned_model.add(Flatten())
    tuned_model.add(Dense(64, activation='relu'))
    tuned_model.add(Dropout(dropout_rate))
    tuned_model.add(Dense(1, activation='sigmoid'))

    optimizer = keras.optimizers.Adam(learning_rate=learning_rate)
    tuned_model.compile(loss='binary_crossentropy', optimizer = optimizer, metrics=['accuracy'])

    return tuned_model
